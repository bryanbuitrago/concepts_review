RESTFul Routing

A table of all 7 RESTful routes

Name Path HTTP Verb Purpose
Index /posts GET Lists all posts
New /posts/new GET Show new post form
Create /posts POST Create a new post, then redirect somewhere
Show /posts/:id GET Show info about one specific post
Edit /post/:id/edit GET Show edit form for one post
Update /posts/:id PUT Update a particular post, then redirect somewhere
Destroy /posts/:id DELETE Delete a particular post, then redirect somewhere
